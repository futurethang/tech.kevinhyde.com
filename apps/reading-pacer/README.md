# ReadPace

A progressive web app to track your reading speed.

## Features

- **Page Timer**: Tap to count pages as you read
- **Pace Tracking**: See your average time per page and last page time
- **Time Estimates**: Calculate how long remaining pages will take
- **Session Saving**: Save and export your reading sessions
- **Wake Lock**: Keep screen on while reading
- **Offline Support**: Works without internet connection
- **Keyboard Shortcuts**: Space (page), P (pause), R (reset)

## Usage

1. Tap the large page counter to mark each page as you finish
2. View your average and last page pace in the stats
3. Enter a number of pages to estimate reading time
4. Use the menu to save sessions or export data

## Install as App

On mobile, use "Add to Home Screen" to install as a standalone app.
