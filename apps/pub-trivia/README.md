# New Year's Eve Pub Trivia 2025

A family-friendly pub trivia game designed for multi-generational New Year's Eve parties.

## Features

- **3 Rounds**: Warm-up, Through the Decades, and Lightning Round
- **Multiple Question Types**: Open answer, multiple choice, musical, riddles, and team challenges
- **Team Scoring**: Track points for multiple teams with live scoreboard
- **Timer**: Configurable time limits per question
- **Host Hints**: Built-in hints to help facilitate the game
- **Fun Facts**: Interesting tidbits revealed with each answer
- **Resume Support**: Game state is saved locally

## Question Categories

### Round 1: The Warm-Up (10 questions, 2 pts each)
General knowledge mix - geography, science, history, music, pop culture, food, literature, brain teasers, sports, and nature.

### Round 2: Through the Decades (10 questions, 3 pts each)
Pop culture spanning the 1960s through 2020s, with something for every generation.

### Lightning Round (15 questions, 1 pt each)
Fast-paced quick fire questions. First to shout the answer wins!

## How to Play

1. **Setup**: Enter team names (minimum 2 teams)
2. **Read Questions**: Host reads each question aloud
3. **Timer**: Teams have the displayed time to discuss and answer
4. **Reveal**: Click "Reveal Answer" to show the correct answer
5. **Award Points**: Click team name to award points
6. **Continue**: Navigate through questions and rounds

## Keyboard Shortcuts

- **Space/Enter**: Reveal answer
- **Arrow Right**: Next question (after reveal)
- **Arrow Left**: Previous question
- **S**: Show scoreboard

## Editing Questions

All questions are stored in `trivia-questions.json`. You can edit this file to:
- Change questions or answers
- Adjust difficulty
- Add/remove questions
- Modify host hints and fun facts

## Files

- `index.html` - Main game interface
- `styles.css` - Styling
- `script.js` - Game logic
- `trivia-questions.json` - All questions and answers (editable!)
