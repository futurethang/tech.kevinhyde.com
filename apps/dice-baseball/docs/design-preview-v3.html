The container has a black background, and 1px grid gaps let the black show through as a uniform slim stroke. Because grid gap is shared space (not per-element borders), there's no doubling where panels meet â€” every junction is exactly 1px. The blue panels get halftone dot overlays.

Design Principles â€” v3 Summary

Color Blocks, Not Borders

Frames are structural color panels in a CSS grid. 1px grid gap on a black container = uniform slim stroke everywhere, no doubling at junctions. Not CSS border properties.

Zero Radius, Always

border-radius: 0 on everything. Cards, buttons, badges, inputs, containers. Only exception: player avatar circles (intentional focal contrast).

Halftone, Not Gradient

Dot patterns and stipple textures where transition is needed. No smooth CSS gradients anywhere. References the offset printing on actual baseball cards.

Script = Celebration

Pacifico for big moments and display text (Home Run!, team banners, Roll Dice). Oswald for structural elements (player names, headings). The font shift IS the reward.

Dot Matrix Scoreboard

Doto at weight 600-800 for all score numbers. The dot construction of the font plus the gold color and glow sells the vintage stadium marquee.

Hard Shadow Physics

Interactive elements have solid offset shadows that compress on press and expand on hover. The physical metaphor makes tapping feel consequential.
