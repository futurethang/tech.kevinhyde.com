Design Principles ‚Äî v2 Summary

FLAT, NOT GRADIENT

Solid color blocks only. Where texture is needed, use halftone dot patterns or stipple overlays ‚Äî never smooth CSS gradients.

SQUARE, NOT ROUND

Border radius: 2px max on containers. Cards, buttons, badges, inputs ‚Äî all hard-edged. Circles only for player avatar initials (a deliberate contrast).

HARD SHADOW, NOT SOFT

Drop shadows are solid-color offsets (4-5px), never blurred. They create depth through geometry, not diffusion. Interactive elements compress the shadow on press.

THICK BORDERS, NOT THIN

Borders are structural (3-5px), colored, and intentional ‚Äî like the card frames in the reference material. No 1px gray hairlines.

WARM, NOT COOL

Text is warm cream (#e8e0d4), not cool white. Surfaces are deep warm navy, not neutral gray. Gold accent over green as the primary color.

CELEBRATION HAS VOICE

Big moments (home runs, wins) use the Righteous jersey script font. Normal UI uses Oswald. The font shift IS the celebration ‚Äî like the "Record Breaker" card treatment.

Animation Direction (spec only ‚Äî punchy & rewarding)

DICE ROLL

Button hard-shadow compresses to 0 on press. Dice emoji shakes 400ms. Result number stamps in with a bounce overshoot (spring, no ease-in-out).

HOME RUN

Gold flash border pulse. "HOME RUN!" in Righteous stamps in large, scales from 1.5‚Üí1.0 with spring. Halftone burst pattern radiates outward briefly.

RUN SCORED

VT323 score number bumps (scale 1.4x‚Üí1.0, 150ms spring). Gold glow pulse on total. Brief dot-matrix "scan line" flickers across the number.

STRIKEOUT / 3RD OUT

Out indicator stamps red with a hard pop (scale 1.5x‚Üí1.0). On 3rd out: bases clear with a quick wipe, inning text slides/swaps with a hard cut ‚Äî no fade.
