# Dice Baseball

A two-player dice baseball game PWA. Roll dice and play ball!

## How to Play

1. Roll two dice - the total determines the result
2. Take turns batting (Visitor vs Home team)
3. 3 outs ends your half-inning
4. Play 9 innings to determine the winner!

## Dice Chart

| Roll | Result |
|------|--------|
| 2 | Home Run! |
| 3 | Triple |
| 4 | Walk |
| 5 | Single |
| 6 | Ground Out |
| 7 | Fly Out |
| 8 | Ground Out |
| 9 | Single |
| 10 | Double |
| 11 | Strikeout |
| 12 | Home Run! |

## Features

- Mobile-first PWA (installable)
- Visual baseball diamond with runners
- Full scoreboard with inning-by-inning tracking
- Works offline once loaded
