What is Life OS Agent?

Life OS Agent is a personal scheduling assistant that uses Claude to manage time allocation across all life domains‚Äîwork, health, creative pursuits, skill development, and maintenance.

Key Features

Natural Language Chat

"Plan my week" or "Add a 3D printing session this weekend" - the agent understands context and constraints.

Persistent Memory

Remember preferences, patterns, and history. "You've skipped Thursday workouts 3 times this month."

Priority-Based Scheduling

Define what's critical vs. nice-to-have. The agent knows what to cut when time is tight.

Gap Analysis

Find unscheduled time and get suggestions on what to fill it with based on your goals.

Human Override

Edit your calendar directly anytime. The agent adapts to your changes.

Weekly Reports

"How much time did I spend on music last month?" - track your time allocation over time.

Technology Stack

Language: Python 3.11+
LLM: Claude API (Anthropic)
Memory: Mem0 (self-hosted with Docker)
Calendar: Google Calendar API (OAuth2)
Config: YAML files
Interface: Rich CLI

Open Source
