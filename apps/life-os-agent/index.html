Life OS Agent

AI-powered personal scheduling assistant

Calendar is the UI. Memory is the work log. Chat is the interface.

What It Does

Natural Language

Schedule, query, and plan using conversational commands

Smart Scheduling

Respects priorities, preferences, and time constraints

Week Planning

Generates proposals based on activity requirements

Memory

Remembers decisions and stated preferences over time

Quick Start

cd apps/life-os-agent
python -m venv .venv
source .venv/bin/activate
pip install -e .

# Configure
cp .env.example .env
# Edit .env with your API keys

# Run
lifeos chat        # Interactive CLI
lifeos serve       # API server
lifeos telegram    # Telegram bot
Backend Service
